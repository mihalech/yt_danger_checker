# YouTube Danger Checker

## Описание

**YouTube Danger Checker** — инструмент для поиска трендовых видео на YouTube и автоматической оценки потенциально опасного контента.

Проект объединяет поиск видео через YouTube Data API, анализ речи, видеоряда, звуковых событий и ключевых слов в единую систему оценки опасности контента.

---

## Возможности

### Поиск видео

- Поиск по ключевым словам
- Поддержка Shorts и обычных видео
- Фильтрация по времени публикации
- Фильтрация по длительности
- Расчёт собственного Trend Score

### Анализ названия

Поиск потенциально опасных слов:

- Оружие
- Насилие
- Терроризм
- Самоубийства
- Взрывы
- Военные конфликты

Поддерживаются русский и английский языки.

### Анализ речи

Используется Faster-Whisper.

Возможности:

- Распознавание речи
- Автоматическое определение языка
- Перевод на английский язык
- Поиск опасных слов и выражений
- Вывод найденных причин срабатывания

### Анализ видеоряда

Используются:

- YOLO (Ultralytics)
- NudeNet

Позволяет обнаруживать:

- Оружие
- Ножи
- Потенциально опасные объекты
- NSFW-контент

### Анализ звуков

Используется YAMNet.

Позволяет обнаруживать:

- Выстрелы
- Взрывы
- Крики
- Драки
- Другие опасные звуковые события

---

## Итоговая оценка

Финальный рейтинг формируется из:

- Рейтинга опасности названия
- Рейтинга опасности речи
- Рейтинга опасности видеоряда
- Рейтинга NSFW-контента
- Рейтинга опасных звуков

Категории:

- Safe
- Suspicious
- 18+

## Установка

Установить зависимости:

```bash
pip install -r requirements.txt
```

Установить:

- FFmpeg
- Deno

---

## Настройка

Создать файл `.env`:

```env
YOUTUBE_API_KEY=ВАШ_API_КЛЮЧ
```

Для доступа к видео, требующим авторизацию:

```text
cookies.txt
```

---

## Запуск

```bash
python yt_api.py
```

После запуска можно:

- Указать запрос
- Выбрать Shorts или обычные видео
- Ограничить время публикации
- Включить анализ опасного контента
